# XYLink WebRTC SDK Demo

小鱼易连 WebRTC SDK Html+Javascript Demo 项目。

此项目使用到的技术栈：Html+Javascript

## 准备工作

1. 注意在`./index.html`中配置第三方登录所需的

- clientId
- clientSecret
- extId
- confNumber

## 运行

### 安装依赖

```bash
$ yarn install
or
$ npm install
```

### 运行

```bash
$ npx live-server
```

启动后使用浏览器打开对应网页，注意请使用 http://localhost:{port}访问

例如：`http://localhost:8080/`
